# AZ-104-Storage-Administration-Lab

## Scenario
I configured secure Azure Storage for a small business environment, including private blob storage, lifecycle management, SAS access, network restrictions, and Azure Files.

## Skills Demonstrated
- Azure Storage account creation
- Secure transfer and TLS configuration
- Blob containers and access tiers
- Blob soft delete and versioning
- Lifecycle management
- Shared Access Signatures
- Storage firewall rules
- Azure file shares
- File share snapshots
- Azure Files soft delete

## Architecture
- Resource group: rg-az104-storage-lab
- Storage account: Standard_LRS StorageV2
- Blob container: reports
- File share: teamshare
- Region: Central US
